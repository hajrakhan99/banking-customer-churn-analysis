# 🏦 Banking Customer Churn Analytics

An end-to-end Data Analytics project focused on identifying customers at risk of leaving a bank (customer churn). This project combines ETL, data preprocessing, SQL analysis, Machine Learning, and interactive Power BI dashboards to generate actionable business insights and improve customer retention strategies.

---

## 📌 Project Overview

Customer churn is a major challenge in the banking industry. Retaining existing customers is often more cost-effective than acquiring new ones. This project analyzes customer behavior, identifies churn patterns, and builds predictive models to help banks take proactive retention measures.

---

## 🎯 Objectives

- Analyze customer demographics and banking behavior.
- Identify key factors influencing customer churn.
- Build an ETL pipeline for data preparation.
- Perform SQL-based data analysis.
- Create interactive Power BI dashboards.
- Develop a Machine Learning model for churn prediction.
- Generate business recommendations for customer retention.
